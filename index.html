Instrucciones para usar cURL con la API

1. Acortar una URL

curl -X POST  https://acortadorphp-production.up.railway.app/api.php \
     -H "Content-Type: application/json" \
     -d '{"url": "https://ejemplo.com", "idUsuario": 1}'
  
curl -X POST  https://ojglez.com/api.php \
     -H "Content-Type: application/json" \
     -d '{"url": "https://ejemplo.com", "idUsuario": 1}'

2. Eliminar una URL

curl -X DELETE  https://acortadorphp-production.up.railway.app/api.php \
     -H "Content-Type: application/json" \
     -d '{"slug": "abc123"}'
